Simulation Game

How to Play:

UH OH!! The painter system is malfunctioning and some components are being painted purple! Fortunately a newly installed robotic arm can automatically pull these scrap items from Line A to prevent them from going to shipping.

However we need your help on Line B to pull the purple components into the scrap containment area. If you accidentally click a green box or a purple component is allowed to go to shipping, the game ends. Warning...As the scrap numbers increase, the conveyors may speed up to make up for the lost products.

See if you can keep up with the robot!
